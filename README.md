# From Passive to Active — A Practical Investing Course

## About This Repo

This is a self-directed investing course designed for a specific learner profile:

- **Platform:** Flexible — currently exploring Questrade, Wealthsimple, and others
- **Existing accounts:** Multiple TFSAs and an RRSP, all invested in low-to-medium risk diversified holdings
- **TFSA/RRSP status:** Tracking contribution limits, within room on both
- **Current self-directed positions:** XEQT (~$100–300), VFV or similar (~$100–300), Bitcoin (~$100–300), Lightspeed Commerce / LSPD (~$100–300)
- **Goal:** Level up from passive/diversified investing to confident, active portfolio management — grow existing capital more aggressively while understanding the risks
- **Time available:** 1–3 hours per week
- **Experience level:** Intermediate — has accounts, has bought ETFs and individual stocks, familiar with basic investing concepts

This repo is structured to be worked through dynamically with Claude Code. Each module is a standalone markdown file. The learner and Claude Code can open any module, ask questions, run exercises, and build on top of the content.

---

## How to Use This With Claude Code

1. Clone this repo locally or open it in your Claude Code environment.
2. Start with `module-01-foundation.md`.
3. At the end of each module there is an **Exercise** section. Complete it before moving to the next module.
4. Use `progress-tracker.md` to log your trades and reflect on decisions.
5. Ask Claude Code anything as you go — it has full context from all files in this repo.

## Core Principles

- Never invest money you cannot afford to lose.
- A loss is tuition. A panic sell is failing the exam.
- Protect capital first. Grow it second.
- Track every decision and the reasoning behind it.
- Patience plus research beats impulse every time.

## The Honest Math

You already have a diversified base. The goal now is to actively manage and grow your portfolio. Index ETFs alone return roughly 10% per year. To meaningfully outperform that, the strategy combines smarter stock selection, selective options trades, and disciplined crypto management. This course teaches how to do that intelligently — not recklessly.
